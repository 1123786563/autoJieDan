### 6.2 部署建议

   ```bash
   mkdir -p deploy/prometheus deploy/grafana/provisioning deploy/logstash/pipeline deploy/logstash/config
   ```

2. **设置环境变量**:
   ```bash
   cp .env.example .env
   # 生成安全密钥
   INTERAGENT_SECRET=$(openssl rand -hex 32)
   GRAFANA_ADMIN_PASSWORD=$(openssl rand -hex 16)
   ```

3. **验证 Docker 构建**:
   ```bash
   docker-compose build
   docker-compose up -d
   ```
