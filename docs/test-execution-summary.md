## 5. 问题清单

### P0 (紧急)

1. **Automaton 覆盖率过低 (0.69%)**
   - 原因：vitest 与 coverage-v8 版本冲突
   - 修复：降级 coverage-v8 至 2.x 版本
   - 影响：无法准确评估测试覆盖

2. **Nanobot Adapter 测试失败 (13 个)**
   - 原因：配置问题
   - 修复：修复测试配置
   - 影响：协议适配层无法验证

### P1 (高优先级)

3. **deploy/ 目录缺失**
   - 需创建：`deploy/prometheus/`, `deploy/grafana/`, `deploy/logstash/`
   - 影响：生产监控配置不完整

4. **npm audit 不可用**
   - 原因：镜像源不支持审计 API
   - 修复：配置官方 npm 源用于审计
